### Colors
The website uses CSS variables for easy color customization. Edit the following variables in `styles.css`:

```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1d4ed8;
    --accent-color: #3b82f6;
    --text-color: #1f2937;
    --light-bg: #f3f4f6;
    --white: #ffffff;
}
```
